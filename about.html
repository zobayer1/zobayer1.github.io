Hi! This is {{site.author}}.

Currently working as a Solution Architect at Tiger IT Bangladesh Ltd. Apart from my day job, I am a Super Tester for Wargaming Asia. Previously I have coached at several code camps and institutes. I have participated in problem setting and judging for several ACM-ICPC regional contests and national level programming contests in Bangladesh.

As a software developer, I mostly work with Python and C++. I have worked briefly with Java and C# on a few projects. As a solution architect, my job is to desgin, decribe and manage the life cycle of a software solution, research new technologies and find ways to apply them to solve software problems. I am always eager to learn and implement industry best-practices, and I have picked up a few ideas along the way which I am eager to share with the world.

I am a firm believer of open source software development. Currently I am working on an open source project named ElasticMigrate, which is a Flyway like tool for Elasticsearch. Although, as usual, I am running behind schedule. Outside of programming, I like to travel, listen to good music, play video games, but most importantly, I love to ride my bike.

Software Engineering is a fascinating world where learning never stops. This is my blog site where I intend to share things I am learning, discuss about the problems and challenges I face when developing a piece of software, new and interesting ideas, coding tips and tricks, and may be even some opinionated useless banters. This site is hosted via GitHub pages. If you want to improve anything, please feel free to head over to GitHub repository for the site, create a PR or open an issue.
